# AI Agent Preferences

## Communication Style

### Chat Mode
- **Do not provide**: Summaries of changes, explanations of how things work, or other narrative details
- **Just do**: Make the requested changes and confirm completion
- Keep responses concise and to the point
- Only provide technical details if explicitly asked

### Exception
- Provide explanations when clarification is needed before proceeding
- Ask questions if the request is ambiguous or requires decision-making
